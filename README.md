# pkscl-email
pkscl 회원가입 시 학교 이메일 인증을 위해 발송되는 학교 이메일 인증 카드와,
사용자가 비밀번호 찾기를 신청할 시에 본인 인증을 위해 발송되는 비밀번호 변경 카드입니다.
